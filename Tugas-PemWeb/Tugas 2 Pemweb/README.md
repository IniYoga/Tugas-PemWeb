# Portfolio Sederhana

# Deskripsi
Project ini merupakan website portfolio sederhana yang saya buat menggunakan HTML dan CSS, website ini berfungsi untuk menampilkan informasi pribadi, layanan, dan kontak.

# Fitur
- Navigasi menu (Home, About, Service, Contact)
- Section About Me
- Section Service
- Section Contact
- Smooth scroll antar halaman
- Hover effect pada menu dan tombol
- Layout menggunakan Flexbox

# Teknologi yang Digunakan
- HTML
- CSS

# Struktur Halaman
1. Home
2. About
3. Service
4. Contact
